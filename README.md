## Overview

Aqua is a take on the streaming service, Hulu. It hosts movie content.

## Configuration

 ### Backend

  * PostgreSQL Database

  * Ruby on Rails
### Frontend

  * React.js with Redux

  * JavaScript

  * HTML5
  
  * CSS3
  
 ## Features
 
 ### User Authentication
  *Users are able to sign in with a unique e-mail and password combination
  
  *A user's password is encrypted using a BCrypt algorithm
